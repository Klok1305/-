# Батч 48: Определение тензоров

## Список билетов

**Билет 48.** Определение тензоров, как полилинейных функций и как наборов координат. Доказательство эквивалентности двух определений. Размерность векторного пространства тензоров типа $(p,q)$.

---

## 1. Смысл билета

Тензор — обобщение векторов, ковекторов, операторов, билинейных форм. Есть два эквивалентных определения: геометрическое (полилинейная функция) и координатное (набор чисел, преобразующихся по определённому закону). Двойственность этих подходов — суть всей тензорной алгебры.

---

## 2. Карта билета

- Что такое полилинейная функция типа $(p,q)$ (Определение 6.1.1 Панова).
- Координаты тензора: компоненты $T^{i_1\ldots i_p}_{j_1\ldots j_q}$.
- Тензорный закон преобразования при замене базиса (Определение 6.2.1 Панова).
- Эквивалентность двух определений.
- Размерность $T^p_q(V) = n^{p+q}$.

---

## 3. Уровень 1 — минимум для тройки

### Определение через полилинейные функции

**Определение 6.1.1 (Панов).** Полилинейной функцией типа $(p,q)$ называется функция

$$\mathcal{T}: \underbrace{V^*\times\cdots\times V^*}_{p} \times \underbrace{V\times\cdots\times V}_{q} \to \mathbb{k}$$

(от $p$ ковекторных и $q$ векторных аргументов), линейная по каждому аргументу.

Пространство таких функций обозначается $P^p_q(V)$.

**Примеры:**
- $p=0, q=1$: линейная функция (ковектор) $\xi \in V^*$.
- $p=1, q=0$: вектор $v \in V$ (как функционал $\xi \mapsto \xi(v)$ на $V^*$).
- $p=0, q=2$: билинейная форма.
- $p=1, q=1$: линейный оператор (по Предложению 6.1.3 Панова: $\mathcal{T}_\mathcal{A}(\xi, v) = \xi(\mathcal{A}(v))$).

### Координатное определение

Зафиксируем базис $e_1, \ldots, e_n$ в $V$ и двойственный базис $\varepsilon^1, \ldots, \varepsilon^n$ в $V^*$ ($\varepsilon^i(e_j) = \delta^i_j$).

**Компоненты тензора:** для $\mathcal{T} \in P^p_q(V)$ определим числа

$$T^{i_1\ldots i_p}_{j_1\ldots j_q} := \mathcal{T}(\varepsilon^{i_1}, \ldots, \varepsilon^{i_p}, e_{j_1}, \ldots, e_{j_q}).$$

По полилинейности $\mathcal{T}$ полностью восстанавливается из компонент:

$$\mathcal{T}(\xi^1, \ldots, \xi^p, v^1, \ldots, v^q) = \xi^1_{i_1} \cdots \xi^p_{i_p}\, v^{j_1}_1 \cdots v^{j_q}_q\; T^{i_1\ldots i_p}_{j_1\ldots j_q}$$

(суммирование по повторяющимся индексам, обозначения Эйнштейна).

**Закон преобразования.** Пусть $C = (c^i_{i'})$ — матрица перехода от базиса $e$ к базису $e'$ (т.е. $e_{j'} = c^j_{j'} e_j$, $\varepsilon^{i'} = c^{i'}_{i} \varepsilon^i$). Тогда компоненты тензора в новом базисе:

$$T^{i'_1\ldots i'_p}_{j'_1\ldots j'_q} = c^{i'_1}_{i_1} \cdots c^{i'_p}_{i_p}\; c^{j_1}_{j'_1} \cdots c^{j_q}_{j'_q}\; T^{i_1\ldots i_p}_{j_1\ldots j_q}.$$

**Определение 6.2.1 (тензор координатный).** Тензором типа $(p,q)$ называется соответствие

$$\{\text{базисы в } V\} \mapsto \{\text{наборы } n^{p+q} \text{ чисел } T^{i_1\ldots i_p}_{j_1\ldots j_q}\},$$

удовлетворяющее закону преобразования выше.

### Эквивалентность двух определений

Любая полилинейная функция $\mathcal{T} \in P^p_q(V)$ задаёт тензор через компоненты $(6.1)$, и обратно — тензор задаёт полилинейную функцию через формулу выше. Это соответствие — изоморфизм:

$$P^p_q(V) \cong T^p_q(V).$$

Это обобщает знакомое: матрица линейного оператора в базисе ↔ сам оператор.

### Размерность

$$\dim T^p_q(V) = n^{p+q}, \quad n = \dim V.$$

Потому что каждый из $p+q$ индексов независимо принимает $n$ значений.

---

## 4. Уровень 2 — для четвёрки

**Мнемоника для закона преобразования:**
- Верхние индексы (контравариантные) преобразуются матрицей $C^{-1}$ (коэффициенты $c^{i'}_{i}$, строка базисной матрицы — верхний индекс «живёт» в $V$).
- Нижние индексы (ковариантные) преобразуются матрицей $C$ (коэффициенты $c^j_{j'}$, столбец базисной матрицы — нижний индекс «живёт» в $V^*$).

Проверка: вектор $v = v^i e_i$, при смене базиса $v^{i'} = c^{i'}_i v^i$ — это матрица $C^{-1}$, соответствует верхнему индексу.

**Пример (тензор типа $(1,1)$ = оператор):**

$$T^i_j = \mathcal{T}(\varepsilon^i, e_j) = \varepsilon^i(\mathcal{A}(e_j)) = A^i_j,$$

т.е. компоненты тензора типа $(1,1)$ — это матрица оператора. Закон преобразования $(T')^{i'}_{j'} = c^{i'}_i T^i_j c^j_{j'}$ — это в точности закон $A' = C^{-1}AC$.

**Метрический тензор** $g_{ij} = (e_i, e_j)$ — тензор типа $(0,2)$. В ОНБ $g_{ij} = \delta_{ij}$.

---

## 5. Уровень 3 — глубина

**Тензорное произведение пространств** (более абстрактный подход): $T^p_q(V) \cong V^{\otimes p} \otimes (V^*)^{\otimes q}$. Универсальное свойство тензорного произведения.

**Инвариантность:** само значение $\mathcal{T}(\xi^1,\ldots,\xi^p, v^1,\ldots,v^q)$ не зависит от базиса — базис нужен только для компонент.

---

## 6. Доказательства

**Эквивалентность (скелет):**
1. Дано $\mathcal{T} \in P^p_q(V)$ → определяем $T^{i_1\ldots i_p}_{j_1\ldots j_q} = \mathcal{T}(\varepsilon^{i_1},\ldots,e_{j_q})$.
2. При смене базиса вычисляем $T'^{i'_1\ldots}_{j'_1\ldots}$ подстановкой $\varepsilon^{i'} = c^{i'}_i \varepsilon^i$, $e_{j'} = c^j_{j'} e_j$ — получаем закон преобразования (из полилинейности).
3. Обратно: из закона преобразования конструируем $\mathcal{T}$ независимо от базиса (значение на любых $\xi, v$ раскладываем по базису и берём соответствующую линейную комбинацию $T^{\ldots}_{\ldots}$).

**Размерность:** базис $P^p_q(V)$ состоит из тензоров $\varepsilon^{j_1} \otimes \cdots \otimes \varepsilon^{j_q} \otimes e_{i_1} \otimes \cdots \otimes e_{i_p}$ (всего $n^{p+q}$ штук).

---

## 7. Примеры и контрпримеры

**Пример 1 (тензор типа (0,0)):** скаляр $\lambda \in \mathbb{k}$ — не меняется при замене базиса.

**Пример 2 (тензор типа (0,1)):** ковектор $\xi = \xi_i \varepsilon^i$. Компонента $\xi_j = \xi(e_j)$. При смене базиса: $\xi'_{j'} = c^j_{j'} \xi_j$ — нижний индекс.

**Пример 3 (тензор типа (1,0)):** вектор $v = v^i e_i$. Компонента $v^i = \varepsilon^i(v)$. При смене базиса: $v'^{i'} = c^{i'}_i v^i$ — верхний индекс.

**Контрпример:** координаты вектора **сами по себе** — не тензор (зависят от базиса), тензором является сам вектор с законом преобразования координат.

---

## 8. Типичные ошибки

- Путать верхние и нижние индексы в законе преобразования (верхние — с $c^{i'}_i$, нижние — с $c^j_{j'}$).
- Думать, что $T^p_q(V)$ — это пространство матриц размера $p \times q$ (нет: это $n^{p+q}$-мерное пространство).
- Не оговаривать, что полилинейность — это линейность по каждому аргументу при фиксации остальных.

---

## 11. Мини-конспект

**Тензор типа $(p,q)$** — два эквивалентных определения:
1. **Геометрическое:** полилинейная функция $\mathcal{T}: (V^*)^p \times V^q \to \mathbb{k}$.
2. **Координатное:** набор $T^{i_1\ldots i_p}_{j_1\ldots j_q}$ (по одному набору на каждый базис), преобразующихся по закону $T'^{i'_1\ldots i'_p}_{j'_1\ldots j'_q} = c^{i'_1}_{i_1}\cdots c^{i'_p}_{i_p}\, c^{j_1}_{j'_1}\cdots c^{j_q}_{j'_q}\, T^{i_1\ldots i_p}_{j_1\ldots j_q}$.

Размерность $T^p_q(V) = n^{p+q}$. Примеры: вектор = $(1,0)$, ковектор = $(0,1)$, оператор = $(1,1)$, билинейная форма = $(0,2)$.
